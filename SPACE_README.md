# RSI Tools MCP Server

A Gradio app that exposes Relative Strength Index (RSI) tools and a lightweight MCP-compatible interface for stock analysis workflows.

## Included tools

- Fetch recent stock prices
- Compute RSI for a ticker
- Interpret a raw RSI value
- Search the built-in RSI knowledge base
- Ask the RSI agent (when `OPENAI_API_KEY` is configured)
